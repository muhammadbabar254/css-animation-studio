# 🎬 CSS Animation Studio

A simple and powerful CSS Animation Studio built with HTML, CSS, and JavaScript.  
Create and preview animations like fade, slide, bounce, and scale in real-time.

---

## ✨ Features

- 🎬 Live animation preview
- ⚡ One-click animation trigger
- 🎨 Multiple animations (Fade, Slide, Bounce, Scale)
- 🧠 Beginner-friendly code
- 💻 Clean UI design
- 🚀 Lightweight and fast

---

## 📂 Project Structure

```
css-animation-studio/
│── index.html
│── style.css
│── app.js
│── README.md
```

---

## 🚀 How to Use

1. Open `index.html` in browser
2. Select animation type
3. Click **Run Animation**
4. Watch live animation on box
